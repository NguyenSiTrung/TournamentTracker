# Tech Stack: Tournament Tracker

## Frontend
- **Language:** HTML5, CSS3, JavaScript (ES6+)
- **Framework:** None — vanilla JS with Module/IIFE pattern
- **Architecture:** Single-page application with tab-based navigation
- **Styling:** Custom CSS with CSS custom properties, glassmorphism design
- **Fonts:** Google Fonts (Inter)
- **Build Tools:** None (no bundler, no transpiler)

## Backend (Planned)
- **Language:** Python 3.x
- **Framework:** FastAPI
- **API Style:** RESTful JSON API
- **Purpose:** Replace localStorage with server-side persistence

## Storage
- **Current:** localStorage (browser-side)
- **Planned:** Server-side database (TBD — SQLite, PostgreSQL, or similar via FastAPI)

## DevOps / Deployment
- **Version Control:** Git
- **Deployment:** Web frontend + FastAPI backend on server

## Dependencies
- No npm/pip dependencies currently (zero-dependency frontend)
- Google Fonts CDN (Inter)
